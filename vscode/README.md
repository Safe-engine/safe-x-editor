# 🖼️ PSD & PNG Tools for VS Code

A Visual Studio Code extension that streamlines your image workflow by adding right-click Explorer context menu options to:

- ✅ **Create a new PSD file**
- 🛠️ **Edit existing PNG files**

---

## 🚀 Features

- **Create PSD File**  
  Right-click on a folder or in the Explorer and instantly generate a blank `.psd` file for your design work.

- **Export layers PSD File**  
  Right-click on a psd file or in the Explorer and instantly generate a exported layers `.png` file for your design work.

- **Edit PNG File**  
  Right-click any `.png` file to open it in your preferred external editor for quick adjustments.

---
